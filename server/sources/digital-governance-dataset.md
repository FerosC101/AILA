# Digital Governance Policy Dataset

> Research dataset of digital-governance / digital-trade instruments, broken down to
> the **policy-measure** level. Each row is a `(country, act, pillar, policy focus)`
> observation — so a single act appears on several rows when it touches multiple
> measures. Cluster for every row is *Digital governance policies*.
>
> The backend parser ([`server/src/sources.ts`](../src/sources.ts)) is header-driven:
> it reads the column names below, extracts the **URL** column as the scrape target,
> and de-duplicates by URL when building the crawl list (`/sources`). The full
> policy-level rows are exposed at `/policies`.
>
> ⚠️ Transcribed verbatim from the source spreadsheet — some quirks are preserved
> (e.g. a few `References` URLs do not match their act, and a few timeframe strings
> are malformed). Verify before relying on any single row.
